1. Layout clásico con grid-template-areas

Definimos zonas con nombre: header, sidebar, main, aside y footer. Cada elemento se asigna a su zona con grid-area.

/* Definición del grid con áreas nombradas */
.layout-clasico {
  display: grid;
  grid-template-columns: 200px 1fr 1fr 200px;
  grid-template-rows: auto 1fr auto;
  grid-template-areas:
    "header  header  header  header"
    "sidebar main    main    aside"
    "footer  footer  footer  footer";
  gap: 12px;
  min-height: 400px;
}

/* Asignación de cada elemento a su zona */
.lc-header  { grid-area: header; }
.lc-sidebar { grid-area: sidebar; }
.lc-main    { grid-area: main; }
.lc-aside   { grid-area: aside; }
.lc-footer  { grid-area: footer; }

2. Posicionamiento por líneas (grid-column / grid-row)

Cada celda se posiciona indicando en qué línea empieza y termina. Líneas de columna: 1│2│3│4│5. Líneas de fila: 1│2│3│4.

/* Grid de 4 columnas x 3 filas */
.grid-lineas {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(3, 120px);
  gap: 8px;
}

/* Elemento A: ocupa columnas 1-3, fila 1 */
.gl-a { grid-column: 1 / 3; grid-row: 1 / 2; }

/* Elemento C: ocupa columna 1, filas 2-4 (span vertical) */
.gl-c { grid-column: 1 / 2; grid-row: 2 / 4; }

/* Elemento E: ocupa columna 4, filas 2-4 */
.gl-e { grid-column: 4 / 5; grid-row: 2 / 4; }

3. Galería responsiva con auto-fit + minmax()

El grid crea tantas columnas como quepan (mínimo 220px cada una). Al reducir la ventana, las tarjetas se reorganizan automáticamente.

/* Galería auto-responsiva SIN media queries */
.galeria-responsiva {
  display: grid;

  /* auto-fit: crea tantas columnas como quepan */
  /* minmax(220px, 1fr): mínimo 220px, máximo 1fr */
  grid-template-columns: repeat(auto-fit, minmax(220px, 1fr));

  gap: 20px;
}

/* Alineación de elementos dentro de sus celdas */

/* start: elementos pegados arriba-izquierda */
.align-start   { place-items: start; }

/* center: elementos centrados en su celda */
.align-center  { place-items: center; }

/* end: elementos pegados abajo-derecha */
.align-end     { place-items: end; }

/* stretch (default): elementos llenan toda la celda */
.align-stretch { place-items: stretch; }

/* place-items es shorthand de: align-items + justify-items */

5. Grid implícito: grid-auto-rows y grid-auto-flow

Solo definimos 3 columnas. Las filas se crean automáticamente con grid-auto-rows: minmax(80px, auto). Con grid-auto-flow: dense se rellenan huecos.

/* Solo definimos columnas; las filas son implícitas */
.grid-implicito {
  display: grid;
  grid-template-columns: repeat(3, 1fr);

  /* Filas implícitas: mínimo 80px, crecen si hay contenido */
  grid-auto-rows: minmax(80px, auto);

  /* dense: rellena huecos cuando hay elementos de distinto tamaño */
  grid-auto-flow: dense;

  gap: 8px;
}

/* Elementos que ocupan más espacio */
.gi-wide { grid-column: span 2; }  /* 2 columnas */
.gi-tall { grid-row: span 2; }     /* 2 filas */

📋 Resumen de propiedades CSS Grid

Propiedades del contenedor (padre)
Propiedad Descripción Ejemplo
display: grid Activa el contexto grid display: grid;
grid-template-columns Define columnas explícitas repeat(3, 1fr)
grid-template-rows Define filas explícitas 100px auto 1fr
grid-template-areas Nombra zonas del grid "header header" "side main"
gap Espacio entre celdas 16px o 10px 20px
justify-items Alinea celdas en eje X start | center | end | stretch
align-items Alinea celdas en eje Y start | center | end | stretch
grid-auto-rows Tamaño de filas implícitas minmax(80px, auto)
grid-auto-flow Dirección de auto-colocación row | column | dense
Propiedades de los items (hijos)
Propiedad Descripción Ejemplo
grid-column Posición en columnas 1 / 3 o span 2
grid-row Posición en filas 2 / 4 o span 2
grid-area Asigna a un área nombrada header
justify-self Alinea este item en eje X center
align-self Alinea este item en eje Y end
