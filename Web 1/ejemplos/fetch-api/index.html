Fetch API en JavaScript

Ejemplos practicos para aprender a hacer peticiones HTTP con fetch(). Todos los ejemplos usan JSONPlaceholder, una API REST publica y gratuita.

1. GET — Obtener una lista

La peticion mas comun. Pedimos todos los usuarios y los mostramos en pantalla. fetch() hace GET por defecto.

2. GET por ID — Buscar uno solo

Agregamos el ID a la URL para pedir un recurso especifico.

3. POST — Crear un recurso

Enviamos datos en el body de la peticion. Hay que indicar el Content-Type y el metodo POST.

4. Manejo de errores

Ojo: fetch() solo rechaza la promesa si hay un error de red. Un 404 o 500 NO lanza excepcion, hay que revisar response.ok a mano.

5. Async/Await

Misma funcionalidad que .then(), pero con sintaxis mas limpia. Usamos try/catch para manejar errores.
